# AI BI Assistant V1 - 知识库

## 数据分析基础

### 什么是数据分析
数据分析是指通过收集、处理、分析数据，从中提取有价值的信息的过程。

### 常见的图表类型

1. **柱状图(Column Chart)**
   - 用途：用于比较不同类别的数值
   - 示例：各部门销售额对比
   - 优点：易于理解，便于对比

2. **折线图(Line Chart)**
   - 用途：用于显示数据的趋势变化
   - 示例：月度销售额趋势
   - 优点：能清晰展示时间序列数据

3. **饼图(Pie Chart)**
   - 用途：用于显示部分与整体的比例关系
   - 示例：产品类别销售占比
   - 优点：易于理解比例关系

4. **面积图(Area Chart)**
   - 用途：用于显示累积数据的变化趋势
   - 示例：各产品累积销售量
   - 优点：能显示整体趋势和部分贡献

5. **散点图(Scatter Chart)**
   - 用途：用于显示两个变量间的相关性
   - 示例：客户消费额与消费频率的关系
   - 优点：能发现变量间的相关性

## SQL 基础

### SELECT 语句
```sql
SELECT column1, column2 FROM table_name WHERE condition;
```

### 常用聚合函数
- COUNT()：计数
- SUM()：求和
- AVG()：平均值
- MAX()：最大值
- MIN()：最小值

### GROUP BY
用于按某个字段分组统计：
```sql
SELECT category, SUM(amount) FROM sales GROUP BY category;
```

## 业务指标

### 销售指标
- **销售额**：所有销售收入总和
- **销售量**：销售产品的数量
- **销售增长率**：(本期销售额-上期销售额)/上期销售额

### 客户指标
- **客户数**：活跃客户总数
- **客户满意度**：客户评分平均值
- **客户留存率**：继续购买的客户比例
